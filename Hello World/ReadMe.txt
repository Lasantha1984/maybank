1. In order to run the application on a local pc, work on db configuration in application.property file.
2. For db , import self contain greeting sql file or just create manually greeting database in mysql server.
3. All greetings will be inserted at the start of application.
4. Objective of the app is depending on the hour of the day greeting will be differ. Application will display 
   
   Good morning if the hour > 0 and  hour < 12
   Good afternoon if the hour >= 12 and hour <= 15
   Good evening if the hour > 15 and hour <= 19
   Good night if the hour > 19
